# eslint-config-paazmaya

> Shared ESLint configuration between my projects since it is possible

## Usage

Please see [ESLint documentation for Shareable Configs](http://eslint.org/docs/developer-guide/shareable-configs).

```json
{
  "extends": "paazmaya"
}
```
